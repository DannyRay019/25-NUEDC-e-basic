# 🏆 2025 电赛 E 题：基于多传感器融合的智能循迹打击系统

## 🚀 项目概述
本项目专为 **2025 年全国大学生电子设计竞赛 E 题** 基础题部分的设计，实现了一套**双核心协同**的智能机器人系统。系统完美融合了实时运动控制与边缘 AI 视觉处理，完成了从自主循迹、精准转弯到视觉靶心锁定的全流程任务。

## 🛠️ 核心技术架构
系统采用 **异构双主控** 设计，任务解耦，性能最大化：

| 子系统 | 主控芯片 | 核心职责 | 关键技术 |
| :--- | :--- | :--- | :--- |
| **下位机 (运动)** | **MSPM0G3507** (TI) | 四轮循迹、里程计、姿态控制 | MPU6050 陀螺仪积分、差速转弯、状态机循迹 |
| **上位机 (视觉)** | **K230** (Canaan) | 靶心识别、云台追踪、激光打击 | OV5640 图像处理、小孔成像解算、自适应查表控制 |

## 💡 创新亮点
- **双核协同**：MSPM0 保证毫秒级运动响应，K230 专注高算力视觉推理。
- **精准控制**：基于陀螺仪积分的闭环转弯算法，误差控制在 ±1.5° 以内。
- **智能追踪**：独创“区域自适应查表法”，结合距离反馈，实现快速稳定的云台锁定。
- **鲁棒性设计**：多级红外滤波与空心靶心密度筛选算法，抗干扰能力强。
